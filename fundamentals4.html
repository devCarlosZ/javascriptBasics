Javascript Arrays

Javascript arrays are used to store multipe values in a single variable.

    
      var cars = ["Saab", "Volvo", "BMW"];

What is an Array?

An array is a special vairable, which can hold more than one value at time.

If you have alist of itmes (a list of car names, for example), storing the cars in single variables could look like this:

        
          var car1 = "Saab";
          var car2 = "Volvo"; 
          var car3 = "BMW";

However, what if you want to loop through the cars and find a specific one? And what if you had not 3 cars, but 300?

The solutoin is an array!

An array can hold many values under a single name, and you can access he values by referring to an index number.

Creating an Array

Using an array literal is the easiset way to creawte a JavaScript Array.

Syntax:

var array_name = [item1, item2, ...];

Spaces and line breaks are not important. A declaration can span multiple lines:

        
          var cars = [
          "Saab",
          "Volvo",
          "BMW"
          ];

Putting a comma after the last element (like "BMW",) is inconsistent across browsers.

IE 8 and earlier will fail.

Using the Javascript Keyword new

The following example also creates an Array, and assigns values to it:

var cars = new Array("Saab", "Volvo", "BMW");

The two examples above do exactly the same. Ther is no need to use new Array().

For simplicity, and readability and execution speed, use the first one (the array literal method).

Acces the Elements of an Array

You access an array element by referring to the index number.

This statement accesses the value of the first element in cars:

var name = cars[0];
Example
        
          var cars = ["Chevy", "Ford", "Dodge"];
          document.getElementById("demo").innerHTML = cars[0];

Note: Array indexes start with 0.

[0] is the first element. [1] is the second element.

Changing an Array Element

This statement changes the value of the first element in cars:

cars[0] = "Opel";

Example
        
          var cars = ["Lincoln", "Cadillac", "Buick"]
          cars[0] = "Jeep"
          document.getElementById("demo").innerHTML = cars[0];

Access the Full Array

With Javascript, the full array can be accesssed by referring to the array name:

Example
        
          var fruit = ["Orange", "Banana", "Apple"];
          document.getElementById("demo").innerHTML = fruit;

Arrays are Objects

Arrays are special type of objects. The typeof operator in JavaScript returns "object" for arrays.

But, Javascript arrays are best describedas arrays.

Arrays use numbers to access its "elements". In this exammple, person[0] returns John:

Array:
var person = ["John", "Doe", 46];

Obects use names to access its "members". In this example, person.firstName returns John:

Object:
var person = {firstName:"Juan", lastName:"Dough", age:46};

Array Elements Can Be Objects

JavaScript variables can be objects. Arrays are special kinds of objects.

Becuase of this, you can have variables of different types in the same Array.

You can have objects in an Array. You can have functions in an Array. You can have arrays in an Array:

myArray[0] = Date.now;
myArray[1] = myFunction;
myArray[2] = myCars;

Array Properties and Methods

The real strength of JavaScript arrays are the built-in array properties and methods:

Examples

          var y = cars.sort();  // The sort() mehtod sorts array

The length Property

The length property of an array returns the length of an array (the number of array elements).

        
          var veggies = ["Carrots", "Peas", "Broccoli", "Peas"];
          veggies.length; // the length of veggies is 4

The length property is always one more than the highest array index.

Accessing the First Array Element

Example

          var first = fruits[0];

Accessing the Last Array Element

Example
        
          fruits = ["Banana", "Orange", "Apple", "Mango"];
          var last = fruits[fruits.length -1];

Example
