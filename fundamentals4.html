w3schools.com, JavaScript Arrays

Javascript arrays are used to store multipe values in a single variable.

What is an Array?

An array is a special vairable, which can hold more than one value at time.

If you have alist of itmes (a list of car names, for example), storing the cars in single variables could look like this:

        
          var car1 = "Saab";
          var car2 = "Volvo"; 
          var car3 = "BMW";

However, what if you want to loop through the cars and find a specific one? And what if you had not 3 cars, but 300?

The solutoin is an array!

An array can hold many values under a single name, and you can access he values by referring to an index number.

Creating an Array

Using an array literal is the easiset way to creawte a JavaScript Array.

Syntax:

var array_name = [item1, item2, ...];

Spaces and line breaks are not important. A declaration can span multiple lines:

        
          var cars = [
          "Saab",
          "Volvo",
          "BMW"
          ];

Putting a comma after the last element (like "BMW",) is inconsistent across browsers.

IE 8 and earlier will fail.

Using the Javascript Keyword new

The following example also creates an Array, and assigns values to it:

var cars = new Array("Saab", "Volvo", "BMW");

The two examples above do exactly the same. Ther is no need to use new Array().

For simplicity, and readability and execution speed, use the first one (the array literal method).

Acces the Elements of an Array

You access an array element by referring to the index number.

This statement accesses the value of the first element in cars:

var name = cars[0];
Example
        
          var cars = ["Chevy", "Ford", "Dodge"];
          document.getElementById("demo").innerHTML = cars[0];

Note: Array indexes start with 0.

[0] is the first element. [1] is the second element.

Changing an Array Element

This statement changes the value of the first element in cars:

cars[0] = "Opel";

Example
        
          var cars = ["Lincoln", "Cadillac", "Buick"]
          cars[0] = "Jeep"
          document.getElementById("demo").innerHTML = cars[0];

Access the Full Array

With Javascript, the full array can be accesssed by referring to the array name:

Example
        
          var fruit = ["Orange", "Banana", "Apple"];
          document.getElementById("demo").innerHTML = fruit;

Associative Arrays

Many programming lanuguages

Arrays with named indexes are called associateive arrays (or hashes).

JavaScript does not support arrays with named indexes.

In JavaScript, arrays always use numbered indexes.

Example

        
          var person = [];
          person[0] = "John";
          person[1] = "Doe";
          person[2] = 46;
          var x = person.length;
          var y = person[0];

WARNING !!

If you use named indexes, JavaScript will redefined the array to a standard object. After that, some array methods andproperties will produce incorrect results.

Example:

        
          var person = [];
          person["firstName"] = "John";
          person["lastName"] = "Doe";
          person["age"] = 46;
          var x = person.length;     // person.length will return 0
          var y = person[0];

The Difference Between Arrays and Objects

In JavaScript, arrays use numbered indexes.

In Javascript, objects use namded indexes.

Arrays are special kind of objects, with numered indexes.

When to Use Arrays. When to Use Objects.

  • Javascript does not support associative arrays.
  • You should use objects when you want the element names to be strings (text).
  • You should use arrays when you want the element names to be numbers.

Avoid new Array()

There is no need to usethe JavaScript's built-in array constructor new Array().

Use [] instead.

        
          var points = new Array(); // Bad
          var points = [];          // Good

These two different statements both create a new array cotaining 6 numbers:

        
          var points = new Array(40, 100, 1, 5, 25, 10); // Bad
          var points = [40, 100, 1, 5, 25, 10];          // Good

The new keyword only complicates the code. It can also produce some unexpected results:

          var points = new Array(40,100); // Creates an array with two elements (40 and 100);

What if I remove one of the elements?

        
          var points = new Array(40); // Create an arraywith 40 undefined elements !!!

How to Recognize an Array

A common question is: How do I know if a variable is an array?

The problem is that the JavaScript operator typeof returns "object":

        
          var fruits = ["Banana", "Orange", "Apple", "Mango"];
          typeof fruits;    // returns object

The typeof operator returns object because a JavaScript array is an object.

Solution 1:

To solve this problem ECMAScript 5 defines a new method Array.isArray().

        
          Array.isArray(fruits);  // returns true

The problem with this solutionis that ECMAScirpt 5 is not supported in older browsers.

Solution 2:

to solve this problem you can create your own isArray()function:

        
          var myArray = function isArray(x) {
            return x.fruits
          }
          console.log(myArray);

The function above always returns ture if the argument is an array.

Or more precisely: it returns true if the object prototype contains the word "Array".

Solution 3:

The instanceof operator returns true if an object is created by a given constructor:

        
          var fruits = ["Banana", "Orange", "Apple", "Mango"];
          fruits instanceof Array;  //returns true

Converting Arrays to Strings

The JavaScript method toString() converts an array to a string of (comma seperated) array values.

Example
        
          var fruits = ["Banana", "Orange", "Apple", "Mango"];
          console.log(fruits.toString());

Result:

Banana,Orange,Apple,Mango

The join() method also joins all array elements into a string.

It behaves just like toString, but in addition you can specify the seperator:

Example
        
          var fruits = ["Banana", "orange", "Apple", "Mango"];
          console.log(fruits.join(" * "));

Result:

Banana * Orange * Apple * Mango

Popping and Pushing

When you work with arrays, it is easy to remove elements and add new elements.

Popping and Pushing

When you work with arrays, it is easy to remove elements and add new elements.

This is what popping and pushing is:

Popping items out of an array, or pushing items into an array.

Popping

The pop() method removes the last element from an array:

Example
        
          var fruits = ["Banana", "Orange", "Apple", "Mango"];
          fruit.pop();  //Removes the last element ("Mango") from fruits

The pop() method returns the value that was "opped out":

Example
        
          var fruits = ["Banana", "Orange", "Apple", "Mango"];
          var x = fruits.pop(); the value of x is "Mango"

Pushing

The push() method adds a new element to an array (at the end):

Example
        
            var fruits = ["Banana", "Orange", "Apple", "Mango"];
            fruits.push("Kiwi");  // Adds a new eleemnt("Kiwi") to fruits

the push() method returns the new array length:

Example
        
            var fruits = ["Banana", "Orange", "Apple", "Mango"];
            var x = fruits.push("Kiwi", "Watermelon");  // the value of x is 6

Shifting Elements

Shifting is equivalent to popping, working on the first element instead of the last.

The shift() method removes the first array element and "shifts" all other elements to a lower index and retuns the string that was "shifted out":

Example
        
            var fruits = ["Banana", "Orange", "Apple", "Mango"];
            fruits.shift();   // Removes the first elment "Banana" from fruits

The unshift() method adds a new element to an array (at the beggining), and "unshifts" older elements:

Example
        
            var fruits = ["Banana", "Orange", "Apple", "Mango"];
            fruits.unshift("Lemon");    //  Adds a new element "Lemon" to fruits

Changing Elements

Array elements are accessed using their index number:

Array indexes start with 0. [0] is the first array element, [1] is the seconds, [2] is the third...

Example
        
            var fruits = ["Banana", "Orange", "Apple", "Mango"];
            fruits[0] = "Kiwi"  //  changes the first element of fruits to "Kiwi"

The length property provides an easy way to append a new element to an array:

Example

          var fruits = ["Banana", "Orange", "Apple", "Mango"];
          fruits[fruits.length] = "Berry";    // Appends "Berry" to fruits
